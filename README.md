# php-dice #
### php-dice is a PHP module for dice rolling ###

This PHP module is used for dice rolling in PHP. The grammar is taken from this website: http://www.metst√ºbchen.de/cgi-bin/DIESERVER/roll.pl

## Requirements ##

It requires the php-peg module to work (hafriedlander/php-peg)

## Installation ##

Clone the repo from GitHub:
